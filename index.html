HTML

설명

HTML은 웹 페이지의 구조를 정의하는 마크업 언어입니다. HTML은 각 요소를 태그로 정의하며, 태그는 <와 >로 묶입니다. 예를 들어, 제목을 나타내는 요소는 <h1> 태그를 사용합니다. HTML은 웹 개발에서 매우 중요한 언어입니다. 파이썬은 웹 개발 분야에서도 사용되기 때문에 HTML을 이해하고 있다면 파이썬 웹 개발에서도 도움이 됩니다. 파이썬 웹 프레임워크에서 HTML 템플릿을 사용하여 웹 애플리케이션을 구축할 수 있습니다. 또한, 파이썬의 BeautifulSoup 라이브러리는 HTML과 XML과 같은 마크업 언어를 파싱하는 데에 사용됩니다. 따라서 HTML을 이해하고 있다면, 파이썬 웹 개발에서 유용하게 사용할 수 있습니다.

CSS

설명

CSS는 HTML로 작성된 웹 페이지의 스타일을 정의하는 스타일 시트 언어입니다. CSS는 HTML 문서 내에서 <style> 태그를 사용하여 스타일을 정의하거나, 외부 스타일 시트 파일을 사용할 수 있습니다. 기존의 HTML은 웹 문서를 다양하게 설계하고 수시로 변경하는데 많은 제약이 따르는데, 이를 보완하기 위해 만들어진 것이 스타일 시트이고 스타일 시트의 표준안이 바로 CSS입니다. HTML을 이용해서 웹 페이지를 제작할 경우 전반적인 틀에서 세세한 글꼴 하나 하나를 일일이 지정해주어야 하지만, 웹 페이지의 스타일(작성형식)을 미리 저장해 두면 웹 페이지의 한 가지 요소만 변경해도 관련되는 전체 페이지의 내용이 한꺼번에 변경되므로, 문서 전체의 일관성을 유지할 수 있고 작업 시간도 단축된다는 장점이 있습니다. 따라서 웹 개발자들은 보다 풍부한 디자인으로 웹을 설계할 수 있고, 글자의 크기, 글자체, 줄간격, 배경 색상, 배열위치 등을 자유롭게 선택하거나 변경할 수 있으며 유지·보수도 간편하게 할 수 있습니다.

JavaScript

설명

JavaScript는 웹 페이지를 동적으로 제어하기 위한 스크립트 언어입니다. JavaScript를 사용하면 웹 페이지의 요소를 조작하거나, 사용자와 상호작용할 수 있는 다양한 기능을 구현할 수 있습니다. 자바스크립트는 크로스 플랫폼(cross platform), 객체지향 스크립트 언어로 웹페이지의 동작을 담당합니다. 예를 들어 자바스크립트를 이용하면 ‘버튼을 클릭하면 밑에 날짜를 보여줘’라는 식의 명령을 내릴 수 있습니다. 자바스크립트를 이용하면 웹에 풍부한 효과를 넣을 수 있습니다.

python

설명

python은 1991년 네덜란드계 소프트웨어 엔지니어인 귀도 반 로섬이 발표한 고급 프로그래밍 언어로, 플랫폼에 독립적이며 인터프리터식, 객체지향적, 동적 타이핑(dynamically typed) 대화형 언어입니다. 문법이 다른 프로그래밍 언어에 비해 어렵지 않을 뿐 아니라 표현하는 구조도 사람이 대화하는 형식을 이용함으로서 초보자도 쉽게 배울 수 있다는 강점이 있습니다. 또한, 개발자인 귀도 반 로섬이 누구든지 무료로 사용할 수 있도록 대중에 완전 공개한 것도 큰 역할을 했습니다.
